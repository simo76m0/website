02.02.02 WEBSITE

Klik på knappen ovenfor for at se opgaven.

I opgaven 02.01.02 Website blev vi for første gang præsenteret for to af de tre sprog, som websites er opbygget af: HTML og CSS. Vi lærte, at HTML er et opmærkningssprog og bruges til at tildele mening til de enkelte tekstbidder, mens CSS er et kaskade sprog der giver HTML det visuelle udtryk. Uden CSS vil HTML ligne et simpelt Word-dokument. På baggrund af seks udleverede wireframes, tilhørende layout diagrammer, tekst og billeder skulle vi opbygge et website om computere.

Et andet krav i opgaven var, at websitet skulle følge de udleverede layout diagrammer med oplysninger om tags, semantik og attributter. I layout diagrammerne blev der blandt andet skelnet mellem grid og flexbox til siderne. Grid tillader os at strukturere indholdet gennem koordinater i form af rækker og kolonner, mens flexbox gør at vi kan arbejde på tværs af bokse. TIl galleri.html brugte jeg for eksempel et grid med fire kolonner, som gjorde at jeg gennem CSS præcist kunne fortælle hvor de forskellige billeder skulle ligge.

"ET SITE OM COMPUTEREN MED ALT FRA DENS HISTORIE TIL DENS SPECIFIKATIONER OG MINE PERSONLIGE PRÆFERENCER.”

Der var forskellige krav til opgaven. For eksempel skulle sitet validere uden fejl. Hertil installerede jeg udvidelsen W3C Web Validator i VS Code. HTML og CSS er opbygget af forskellige syntakser, også kaldet byggeklodser, der tilsammen udgør “reglerne” om sproget. W3C Web Validator hjælper med at fortælle, om syntaksen er skrevet forkert eller om der mangler noget.
